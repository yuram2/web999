속성 설명
<a> href 특성을 통해 다른 페이지나 같은 페이지의 어느 위치, 파일, 이메일 주소와 그 외 다른 URL로 연결할 수 있는 하이퍼링크를 만듭니다.
<abbr> 태그는 준말 또는 머리글자를 나타냅니다.
<acronym>
<address> 태그는 가까운 HTML 요소의 사람, 단체, 조직 등에 대한 연락처 정보를 나타냅니다.
<applet> 자바 애플릿을 보이게 하는 곳에 쓰입니다.
<area> 이미지의 핫스팟 영역을 정의하고 hyperlink를 추가할 수 있습니다. map 요소 안에서만 사용할 수 있습니다.
<article> 태그는 해당 문서나 페이지 또는 사이트와는 완전히 독립적으로 구성할 수 있는 요소를 정의할 때 사용합니다.
<aside> 문서의 주요 내용과 간접적으로만 연관된 부분을 나타냅니다. 주로 사이드바 혹은 콜아웃 박스로 표현합니다.
<audio>
<b>
<base>
<basefont>
<bdi>
<bdo>
<big>
<blockquote>
<blink>
<body> 문서 본문에 해당하는 콘텐츠를 포함하는 영역입니다.
<br> 줄 바꿈 할때 쓰는 영역 입니다.
<button> 태그는 버튼을 만들 때 설정합니다.
<canvas>
<caption> 표의 제목을 설정합니다.
<center> 태그는 HTML5에서 지원되지 않고 대신 CSS에 사용합니다.
<cite>
<code> 코드를 나타내는 텍스트를 설정합니다.
<col> <colgroup>요소에 속하는 각 열의 속성을 설정합니다.
<colgroup> 표의 열을 묶는 그룹을 설정합니다.
<data>
<datalist>
<dd>
<del>
<details>
<dfn>
<dialog>
<dir>
<div> 문서의 영역을 나누거나 그룹화 할 때 설정합니다.
<dl>
<dt>
<em> 텍스트를 강조할 때 설정합니다.
<embed>
<fieldset> 요소에서 연관된 요소들을 하나의 그룹으로 묶을 때 사용합니다.
<figcaption>
<figure>
<font> 텍스트의 폰트(font), 색상 그리고 크기를 정의할 때 사용합니다.
<footer> footer 요소는 HTML 문서나 섹션(section) 부분에 대한 푸터(footer)을 정의합니다.
<form>
<frame>
<frameset>
<h1> ~ <h6> 섹션의 제목을 설정합니다.
<head> 제목과 스크립트, 스타일 시트, 메타 정보 등을 제공하는 영역입니다.
<header> 태그는 문서나 특정 섹션(section)의 헤더(header)를 정의할 때 사용합니다.
<hgroup>
<hr> 요소간의 주제별 구분을 설정합니다.
<html> 웹 문서의 최상위 요소 문서의 루트를 나타냅니다.
<i>
<iframe>
<img> 웹 문서에 이미지를 넣습니다.
<input> 태그는 사용자 입력정보를 받을 때 설정합니다.
<ins>
<kbd>
<keygen>
<label>
<legend>
<li> 목록의 항목을 설정합니다.
<link> 외부 파일을 연결할 때 설정합니다.
<main> 웹 문서의 주요 콘텐츠 주요 영역을 설정합니다.
<map>
<mark>
<marquee>
<menu>
<menuitem>
<meta> 메타태그는 웹 서버와 웹 브라우저간에 상호 교환되는 정보를 정의하는데 사용합니다.
<meter>
<nav> 내부 페이지 및 외부 페이지에 대한 탐색 링크를 제공하는 페이지 섹션을 설정합니다.
<noframes>
<noscript> 태그는 클라이언트 사이드 스크립트(client-side scripts)를 사용하지 않도록 설정했거나, 스크립트를 지원하지 않는 브라우저를 위한 별도의 콘텐츠를 정의할 때 사용합니다.
<object>
<ol>
<optgroup>
<option>
<output>
<p> 단락을 나타낼 때 설정합니다.
<param>
<picture>
<pre> 지정한 텍스트를 나타내며, HTML에 작성한 내용 그대로 표현합니다.
<pregress>
<q>
<rp>
<rt>
<rtc>
<ruby>
<s>
<samp> 프로그램으로 출력된 결과물을 나타내는 텍스트를 정의합니다.
<script> 스크립트를 정의합니다.
<section> 주제별 그룹의 콘텐츠 섹션을 정의합니다.
<select>
<small>
<source>
<span> HTML 문서에서 인라인 요소(inline-element)들을 하나로 묶을 때 사용합니다.
<strike>
<strong> 텍스트의 중요성을 강조할 때 설정합니다.
<style> 웹 문서에 스타일 정보를 설정합니다.
<sub>
<summary>
<sup>
<svg>
<table> 표를 설정합니다.
<tbody> 표의 본문 영역을 그룹화합니다.
<td> 표의 열(셀)을 설정합니다.
<template>
<textarea>
<tfoot> 태그는 표의 푸터 영역을 그룹화합니다.
<th> 행이나 열에 타이틀을 설정합니다.
<thead> 표의 헤더 영역을 그룹화합니다.
<time>
<title> 브라우저의 제목 또는 탭에 표시되는 문서의 제목을 설정합니다.
<tt>
<u>
<ul> 순서가 없는 목록형 태그를 정의합니다.
<var> 수학적인 변수나 프로그램의 변수를 쓸 때 사용합니다. 일반적으로 기울기체로 표시되지만 브라우저마다 다릅니다.
<video>
<wbr>
