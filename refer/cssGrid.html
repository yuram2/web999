속성 설명
grid 두 방향(가로-세로) 레이아웃 시스템 (2차원)
grid-area Grid 라인으로 둘러싸인 사각형 영역으로, 그리드 셀의 집합입니다.
grid-auto-columns 암시적인 열(Track)의 크기를 정의
grid-auto-flow 아이템이 자동 배치되는 흐름을 결정하는 속성입니다.
grid-auto-rows 암시적인 행(Track)의 크기를 정의
grid-column grid-column-xxx의 단축 속성(열 시작/끝 위치)
grid-column-end 그리드 아이템의 열 끝 위치 지정
grid-column-gap 열 사이의 여백 설정
grid-column-start 그리드 아이템의 열 시작 위치 지정
grid-gap Grid 셀 사이의 간격입니다.
grid-row grid-row-xxx의 단축 속성(행 시작/끝 위치)
grid-row-end 그리드 아이템의 열 끝 위치 지정
grid-row-gap 행 사이의 여백 설정
grid-row-start 그리드 아이템(Item)의 행 시작 위치 지정
grid-template grid-template-xxx의 단축 속성
grid-template-areas 영역(Area) 이름을 참조해 템플릿 생성
grid-template-columns 열(column)의 배치
grid-template-rows 행(row)의 배치
