JAVASCRIPT

자바스크립트를 공부하며 정리한 사이트 입니다.
변수, 상수, 배열, 객체등의 데이터를 저장, 불러오기, 실행하기, 제어하기
내부 함수, 즉시실행 함수, 파라미터 함수, 아규먼트 함수, 재귀 함수, 콜백 함수,
객체생성자 함수, 프로토타입 함수, 객체 리터럴 함수,
등 기초문법들이 정리되어 있습니다.

01

CSS

HTML과 CSS를 공부하며 정리한 페이지 입니다.
기본 문법, 속성 각종 테그, 효과등을 보고 꾸준하게 공부를 하며
기록해 두었습니다.
부족한 부분을 공부하며 새로운 기술들을 업데이트 할 예정입니다.

02

FLEX

레이아웃 프레임을 만들면서 Flex속성을 공부하며 정리한 페이지 입니다.
Flex는 1차원적으로 한 방향 레이아웃 시스템으로
부모 요소인 Flex Container가 Flex의 영향을 받는 전체 공간에
자식 요소인 Flex Item에 설정된 속성에 따라 각각의 아이템들이 지정된 형태로
배치하여 기본적인 레이아웃 틀을 완벽하게 구현 가능합니다.

03

GRID

Grid는 가로행(Row), 세로열(Column) 두 방향의 2차원적 레이아웃 시스템 으로
부모 요소인 그리드 컨테이너에 자식 요소인 그리드 아이템을
그리드 트랙, 그리드 셀, 그리드 라인, 그리드 번호, 그리드 갭, 그리드 영역 등을
설정하여 Flex보다 더 복잡한 레이아웃도 구현이 가능합니다.

04

MOUSE EFFECT

three.js 라이브러리API를 이용하여 마우스 효과를 구현하였습니다.
페이지 좌표 값과 마우스 좌표값, 스크린 좌표값을 통해
마우스 따라다니기, 이팩트, 마우스포인터 변환등을 표현하였습니다.

05

SLIDER EFFECT

여러 이미지 슬라이드 방식을 구현한 페이지 입니다.
가로, 세로 방향의 이미지 슬라이드, 상,하 방향의 슬라이드,
자동 이미지 슬라이드, 연속적인 이미지 슬라이드등을 표현하였습니다.

06

PARALLAX EFFECT

scrolltop값과 offset값으로 스크롤시 나타나는 이벤트 페이지 입니다.
마우스 스크롤시 hover 효과, 가로 스크롤, 세로 스크롤을 구현하였습니다.
메뉴 버튼과 닷 버튼을 추가하여 원하는 곳으로 이동이 가능하게 하였습니다.
