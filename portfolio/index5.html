02

FLEX

레이아웃 프레임을 만들면서 Flex속성을 공부하며 정리한 페이지 입니다.
Flex는 1차원적으로 한 방향 레이아웃 시스템으로
부모 요소인 Flex Container가 Flex의 영향을 받는 전체 공간에
자식 요소인 Flex Item에 설정된 속성에 따라 각각의 아이템들이 지정된 형태로
배치하여 기본적인 레이아웃 틀을 완벽하게 구현 가능합니다.

03

GRID

Grid는 가로행(Row), 세로열(Column) 두 방향의 2차원적 레이아웃
시스템 으로 부모 요소인 그리드 컨테이너에 자식 요소인
그리드 아이템을 그리드 트랙, 그리드 셀, 그리드 라인, 그리드 번호,
그리드 갭, 그리드 영역 등을 설정하여 Flex보다 더 복잡한
레이아웃도 구현이 가능합니다.
